# 🌐 Godwin Binoy — Portfolio

Premium reel-style scrolling portfolio built with **Astro 5** + **Tailwind CSS 3.4**.

## ✨ Features
- 🎬 Reel-style snap scrolling (6 full-screen sections)
- 🌌 Animated particle network background
- 🪟 Glassmorphism dark theme
- 📱 Fully responsive (mobile + desktop)
- ⚡ Zero JS framework overhead (Astro islands)
- 🚀 GitHub Actions CI/CD → GitHub Pages
- ♿ Semantic HTML + ARIA + keyboard nav (↑↓)

## 🛠 Stack
Astro 5 · Tailwind CSS 3.4 · GitHub Actions · GitHub Pages

## 🚀 Dev
```bash
npm install && npm run dev
```

## 📦 Deploy
Auto-deploys on push to `main` via GitHub Actions.
